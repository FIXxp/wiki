# Полезные файлы

## SamopisAuthlib

Быстрая установка аучлибы для fabric & forge выше 1.13.2

- Качаем
- Кидаем в папку mods
- Радуемся, ведь нам не надо думать о сборке аучлибы и читать МНОГА БУКФ на вики  

[SamopisAuthlib-1.0.jar](/files/SamopisAuthlib-1.0.jar)
